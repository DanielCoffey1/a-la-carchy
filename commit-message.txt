feat(battery): add charge limit slider to walker power profile menu

Adds a persistent battery charge limit control directly in the Omarchy
power profile walker menu (battery icon popup in waybar). When a limit
is set, the menu shows a visual bar with the current percentage below
the existing power profile options, and selecting it opens a sub-menu
to adjust the limit via pkexec.

- Add 70% option to battery limit dialog (now 60/70/80/90/100%)
- Create helper script at ~/.config/hypr/scripts/omarchy-battery-limit.sh
  that uses pkexec (GUI-friendly) to set sysfs + udev rule in one call
- Override show_setup_power_menu() in extensions/menu.sh with managed
  block that appends charge limit bar below power profiles
- Auto-cleanup helper and override when limit is set back to 100%
- Update waybar tooltips and send desktop notifications from walker menu
